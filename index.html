OpenShift Mobile

A multi-platform mobile application designed to allow
users to manage and control their OpenShift account.

 

Mobile PaaS

OpenShift is Red Hat's Platform-as-a-Service (PaaS) that allows developers to quickly develop, host, and scale applications in a cloud environment. With OpenShift Mobile developers and operations teams have the ability to create, control and manage the applications hosted on the OpenShift platform.

Take the Tour

See how easy it is to manage your OpenShift account using OpenShift Mobile. Get a first hand look at the well designed views of the login, application, and settings pages. Discover how easy it is to see the details and start, stop, restart, or delete your application all from OpenShift Mobile.

Want More

OpenShift Mobile is open source so you can download or browse the code and build or extend the application from source yourself. Check out the wiki to learn how to get started. There is even a demo you can test drive (currently requires Chrome browser and Ripple plug-in). Fork on GitHub!
